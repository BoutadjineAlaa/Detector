# JSDetector
Helps you detect ENDPOINTS from Javascript.

# INSTALL 

git clone https://github.com/BoutadjineAlaa/Detector.git

cd Detector

pip3 install -r Requirements.txt

# USAGE

python3 JSDetector.py --url https://example.com/file.js --output result.txt
